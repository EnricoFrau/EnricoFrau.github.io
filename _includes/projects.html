Computational Physics Course Project

Simulation of magnetic phase transitions in 2D spin lattices using cellular automata. The computational core is written in Fortran for performance, while Python manages the analysis and visualization layer. I selected this project to highlight the capability to understand when to use low-level languages for heavy computation and how to integrate them effectively with higher-level tools. It also demonstrates numerical methods for studying critical phenomena in statistical mechanics.
